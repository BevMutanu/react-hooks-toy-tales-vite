# Toy Tales - CRUD React Application

## Overview

Toy Tales is a React CRUD application that allows users to interact with a collection of toys. Users can:

- View all toys
- Add new toys
- Like toys
- Delete toys (“Donate to GoodWill”)

This project demonstrates full CRUD operations using a RESTful API powered by json-server.
